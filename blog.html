Question-1: What are the different ways to select an element in the DOM?

Answer : There are several ways to select an element of dom : a. By ID (getElementById), b. By Class Name (getElementsByClassName), c. By Tag Name (getElementsByTagName), d. By CSS Selector (querySelector), e. By CSS Selector (querySelectorAll)

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer : The difference between innerHTML, innerText and textContent are "innerHTML--Gets or sets HTML content, including tags". "innerText--Gets or sets only visible text, affected by CSS". "textContent--Gets or sets all text, including hidden text, but ignores HTML formatting".

Question-4: What is event bubbling in the DOM?

Answer : Event bubbling to the DOM is a mechanism in the DOM where an event triggered on a child element propagates (bubbles up) through its ancestors, all the way to the root element.
